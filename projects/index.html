Portfolio

Welcome to my portfolio! Here you'll find a collection of data analysis projects I have developed for academic, professional, or personal research purposes. Each project was designed to solve real-world problems, extract insights from data, and support evidence-based decision-making.

Projects

Bank Customer Analysis

A denormalized feature table is created using SQL joins and aggregations on client, account, and transaction data. The table enables machine learning models for predicting customer behavior, reducing churn, managing risk, and personalizing offers..

Analysis of Air Disasters from 1919 to 2023

Analyze 25,000 aviation incidents (1919–2023) to identify key trends by country, date, operator, and aircraft type, and visualize findings including a cartogram of incidents per country.

Rome's Green Lungs

Focused on the correlation between air quality and green spaces in Rome, highlighting public parks role in reducing nitrogen dioxide emissions.

Report on stores of an electronics retail chain

Develop an interactive sales report to help managers monitor sales, product performance, promotions, and returns across stores for better decision-making.

Consumer complaints analysis

The goal is to improve the effectiveness of customer complaint analysis by restructuring and customizing a spreadsheet, in order to optimize information access and the decision-making process.

Trading operations analysis

The project transforms a spreadsheet into a dynamic tool for analyzing trading operations, enhancing visualization, automation, and support for data-driven strategic decisions.

Random data and linear regression

The project focuses on creating a realistic dataset using a spreadsheet to provide an effective tool for testing statistical analysis techniques and predictive models.
